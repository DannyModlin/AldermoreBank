# AldermoreBank
Two Lunch and Learn Sessions on Time Series


### Tuesday, July 14th Presentation:

## Time Series Analysis in SAS Studio – Getting Started
This presentation will start with an introduction to time series analysis using SAS Studio. Focus will be placed on using both code aspects as well as any available SAS Studio Steps. Topics included will start with time series creation, identification and autocorrelation. We will start creating models such as exponential smoothing, ARIMAX, and unobserved components models.


### Thursday, August 13th Presentation:

## Time Series Analysis in SAS Studio and Visual Forecasting
In this follow-up presentation, we will finish our discussion of the creation of models such as exponential smoothing, ARIMAX, and unobserved components models. We will then explore the Visual Forecasting interface within SAS Model Studio. We will discuss setting up a project and the creation of models within this interface. A comparison of the SAS Studio and the SAS Model Studio interfaces will be made.
